# rn-yangga-app-ads

react native ads module for yangga

## Installation

```sh
npm install rn-yangga-app-ads
```

## Usage

```js
import * as YanggaAppAds from 'rn-yangga-app-ads';

// App.tsx
// ...
   <YanggaAppAds.Provider>
    ...
   </YanggaAppAds.Provider>

// in your component
// ...
    <YanggaAppAds.Banner />

```
